# 教程简介

:::tip

YunzaiJS 机器人开发框架，可以通过开发模块化应用和集成丰富的生态来扩展机器人的功能。 遵循向下兼容，向上扩展的原则来支持更丰富的接口。它是一款追求代码可维护性、可扩展性、稳定性和高性能，提供对用户友好，对开发者友好的机器人开发框架。

:::

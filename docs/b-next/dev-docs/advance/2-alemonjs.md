# AlemonJS 阿柠檬

:::tip AlemonJS

[AlemonJS](https://github.com/lemonade-lab/alemonjs) ( 发音为 /əˈlemən/ ) 基于 JavaScript 所构建的跨聊天平台开发框架

:::

你可以使用 AlemonJS 环境, 开发聊天机器人、聊天插件、聊天频道、聊天群组等应用。
